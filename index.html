Pocketjs

The mini jQuery-like library

View on GitHub Download .zip Download .tar.gz

It's not the library you deserve, it's the library you need.

A lot of websites load the entire jQuery library, yet only use a subset of the available methods. PocketJS combines some of the most heavily used jQuery DOM methods into a tiny "pocket-sized" 1KB file. (Legal Disclaimer: 1kb AFTER minification. Not guaranteed to fit in your pocket.)

Installation

To use, simply include the script anywhere on your page:

  <script src="pocket.min.js"></script>

Usage

A list of available methods and example usages of each is located on Github.
