Pocketjs

The mini jQuery-like library

Download .zip Download .tar.gz View on GitHub

It's not the DOM manipulation library you deserve, but the DOM manipulation library you need.

In my experience, I've noticed that a lot of websites/projects load the entire jQuery library, yet only use a subset of the available methods. PocketJS combines some of the most heavily used jQuery DOM methods into a "pocket-sized" tiny 1KB file. 'Legal Disclaimer: 1kb AFTER minification. Not guaranteed to fit in your pocket.'

Installation

To use, simply include the script anywhere on your page:

  <script src="pocket.min.js"></script>

Usage

A list of available methods and example usages of each is located on Github.
